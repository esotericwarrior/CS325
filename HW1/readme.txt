Instructions for compiling each program on FLIP.

To compile insertsort.cpp, type the following on the command line:
g++ -std=c++11 insertsort.cpp -o insertsort

To compile mergesort.cpp, type the following on the command line:
g++ -std=c++11 mergesort.cpp -o mergesort

To compile mergeTime.cpp, type the following on the command line:
g++ -std=c++11 mergeTime.cpp -o mergeTime

To compile insertTime.cpp, type the following on the command line:
g++ -std=c++11 insertTime.cpp -o insertTime
